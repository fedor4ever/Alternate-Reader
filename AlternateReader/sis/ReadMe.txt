Djvu Reader for Symbian 9.x- это абсолютно бесплатная программа для просмотра популярного формата djvu на телефонах Symbian OS 9.x

В этой версии выводится только монохромное изображение документа, а также еще не поддерживается touch-interface для S60 5-th edition. Когда проект будет окончен, скорее всего он станет open-source. Всё зависит от наличия свободного времени у автора, а также финансовой поддержки пользователей.


Для нормальной работоспособности необходимо установить Symbian P.I.P.S. - C-runtime library
